

# Task Tracker (CRUD) App Using MySQL Database #


## Requirements ##

1. MySQL Workbench
2. React JS
3. Node JS
4. Express
5. Cors
6. Nodemon
7. Axios

# Features #

1. Server/Back-end API (Creating Tasks, Reading Tasks, Updating Tasks, and Deleting Tasks)
2. Front-end displays Tasks in order.
